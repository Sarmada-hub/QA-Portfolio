# Test Strategy

## Test objectives

Ensure system stability and data consistency.

## Test types

- Functional testing
- Access control testing
- Negative testing
- User acceptance simulation
- Regression testing

## Test scope

- Authentification
- Session management
- Appointment booking
- Appointment cancellation

## Risk Analysis

| Risk | Impact |
|------|--------|
| Double booking | High |
| Unauthorized access | Critical |
| Invalid data acceptance | High |
