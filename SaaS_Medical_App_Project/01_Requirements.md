# Requirements

## Authentification 

REQ-ACC-001  
Users shall access public pages without authentication.

REQ-ACC-002  
Registered users will authenticate using valid credentials.

REQ-ACC-003  
Authentication will fail when credentials are invalid.

REQ-ACC-004  
Password rules will be enforced during account creation.

REQ-ACC-005  
Registration will succed when credentials are valid.

REQ-ACC-006  
Registration will fail when credentials are invalid.

---

## Appointment management

REQ-APT-001
Available appointment will be displayed.

REQ-APT-002
Available appointment won't be dislayed.

REQ-APT-003
User can book and appointment.

REQ-APT-004
User must be able to cancel an existing appointment no later than 24 hours before the appointment.

REQ-APT-005
User is not able to cancel an existing appointment less than 24 hours before the appointment.
